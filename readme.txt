C++언어, visual studio를 기반 OpenFramework를 사용하였음


실행방법:
소스 파일은 총
	main.cpp
	ofApp.cpp
	ofApp.h
	ofWinMenu.cpp
	ofxWinMenu.h
로 구성되어 있음

플랫폼 도구 집합을 버전에 맞게 변경 후 코드 실행

프로젝트를 실행하면 
../bin/data 에 있는 다음 3가지 .maz 파일 중 하나를 선택하여 실행 (미로크기, 보물 개수 차이)
	maze_easy 난이도 쉬움  (미로 크기: 5*5,  보물 개수: 1)
	maze_medium 난이도 보통  (미로 크기: 10*10,  보물 개수: 3)
	maze_difficult 난이도 어려움  (미로 크기: 20*20,  보물 개수: 6)



play방법:

  게임 종료 조건: 보물(핑크색 보석 모형)을 다 획득 후, 도착 지점(미로의 가장 우측 하단 지점)에 도달

  방향키를 이용하여 player 위치 변경 가능

  보물이나 아이템이 있는 위치에 player가 도달하면 자동 획득

  아이템(노란색 ? 박스) 사용 방법 : 
 	아이템을 획득 후 'h'키를 누르면 현재 위치에서 도착지점까지 최단 거리 제공
        (단, 'h' key를 누르고 있는 동안만 볼 수 있음. 기회는 1번)

  게임 종료 조건을 만족하면 playtime을 보여주며 게임 종료

  다시 .maz파일을 불러오면 게임 재시작 가능
